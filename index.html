Features

Everything your
server needs to stay in control

⚡
Real-time logging

Events are captured and posted instantly — no delays, no missed actions. If it happens in your server, LogBot catches it.

🎛️
Web dashboard

Configure every log type from a clean web UI. Toggle events on or off, assign channels — no more slash command clicking.

🔐
Discord OAuth login

Sign in with your Discord account. LogBot only shows servers where you have admin permissions — no token sharing needed.

🎨
Rich embeds

Every log message is a beautifully formatted embed with color-coded severity, user avatars, and all relevant details.

📦
Per-event channels

Route each log type to a different channel. Put bans in mod-logs, voice events in voice-logs — total separation.

🔍
Audit log aware

LogBot cross-references Discord's audit log to surface who performed an action — not just what happened.

Setup

Up and running in 3 steps

01
Invite LogBot

Click "Add to Discord" and select your server. LogBot needs no special permissions beyond what it needs to read events.

02
Log in to the dashboard

Sign in with Discord OAuth. You'll see all servers where you're an admin and LogBot is present.

03
Configure your logs

Toggle each event type on, assign a channel, and hit save. Changes go live instantly — no restart needed.
